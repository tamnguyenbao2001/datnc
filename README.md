# datnc
## ĐATN
1. Cài đặt trước các thư viện 
`pip install -r requirements.txt`

Nhớ cài thêm cuda, cudnn để chạy gpu nha
`https://pytorch.org/get-started/previous-versions/`

2. Run code

```bash

$ python datn.py --device 0

```
